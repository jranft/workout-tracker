WORKOUT TRACKER - DATA SCHEMA
============================

Database Table: workouts
Location: workouts.db (SQLite)

FIELDS SAVED
============

1. id
   Type: INTEGER (PRIMARY KEY, AUTOINCREMENT)
   Description: Unique identifier for each workout record
   Input: Auto-generated
   Calculated: No

2. workout_type
   Type: TEXT (NOT NULL)
   Description: Type of workout performed
   Input: Dropdown selection (e.g., "Indoor Air Bike", "Other 1", etc.)
   Calculated: No
   Required: Yes

3. workout_date
   Type: TEXT (NOT NULL)
   Description: Date the workout was performed
   Format: YYYY-MM-DD
   Input: Date picker
   Calculated: No
   Required: Yes

4. start_time
   Type: TEXT (nullable)
   Description: Time the workout started
   Format: HH:MM (24-hour format)
   Input: Time picker
   Calculated: No
   Required: No

5. end_time
   Type: TEXT (nullable)
   Description: Time the workout ended
   Format: HH:MM (24-hour format)
   Input: Time picker
   Calculated: No
   Required: No

6. duration
   Type: TEXT (nullable)
   Description: Total workout duration in hh:mm:ss format
   Format: HH:MM:SS (e.g., "00:20:05")
   Input: Text input with pattern validation
   Calculated: No
   Required: No

7. decimal_duration
   Type: REAL (nullable)
   Description: Workout duration converted to decimal minutes
   Formula: hours * 60 + minutes + seconds / 60
   Input: Auto-calculated from duration field
   Calculated: Yes (client-side JavaScript)
   Example: "00:20:05" → 20.08 minutes

8. miles
   Type: REAL (nullable)
   Description: Distance traveled in miles
   Input: Number input (step 0.01)
   Calculated: No
   Required: No

9. active_calories
   Type: INTEGER (nullable)
   Description: Active calories burned during workout
   Input: Number input
   Calculated: No
   Required: No

10. avg_heart_rate
    Type: INTEGER (nullable)
    Description: Average heart rate during workout in beats per minute (bpm)
    Input: Number input
    Calculated: No
    Required: No

11. max_heart_rate
    Type: INTEGER (nullable)
    Description: Maximum heart rate during workout in beats per minute (bpm)
    Input: Number input
    Calculated: No
    Required: No

12. created_at
    Type: TEXT (default: current timestamp)
    Description: Timestamp when the record was created
    Format: datetime (auto-generated by database)
    Input: Auto-generated
    Calculated: No

CALCULATED FIELDS (NOT SAVED)
=============================

These are calculated on the client-side and displayed to the user but NOT stored in the database:

- Decimal Duration: Converted from duration (hh:mm:ss → decimal minutes)
- Miles Per Hour: miles / (decimal_duration / 60)
- Minutes Per Mile: MM:SS format of decimal_duration / miles
- Calories Per Minute: active_calories / decimal_duration
- Calories Per Mile: active_calories / miles
- Total Heart Beats: avg_heart_rate * decimal_duration (rounded to integer)

VALIDATION RULES
================

Required Fields:
- workout_type: Must be selected from dropdown
- workout_date: Must be valid date in YYYY-MM-DD format

Optional Fields:
- All other fields are optional

Data Type Validation:
- miles: Positive decimal number (step 0.01)
- active_calories: Positive integer
- avg_heart_rate: Positive integer
- max_heart_rate: Positive integer
- duration: Must match pattern \d{2}:\d{2}:\d{2}
- start_time, end_time: Valid time format HH:MM
