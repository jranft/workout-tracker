# Home Page Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a home page at `/` with a sortable workout history table and a "New Workout" button that navigates to the existing form at `/new`.

**Architecture:** Move the current `GET /` form route to `GET /new`, create a new `GET /` home route that queries all workouts and passes them to a new `home.html` template. After `POST /log` saves, redirect to `/` so the newest entry appears at the top of the table. Sorting is client-side JS on column header clicks.

**Tech Stack:** Flask, SQLite, Jinja2, vanilla JS (no dependencies)

### Task 2: Add home route that queries all workouts

**Files:**
- Modify: `app.py`

**Step 1: Add the home route**

Add this route to `app.py` (above `new_workout`):

```python
@app.route("/", methods=["GET"])
def index():
    conn = get_db()
    try:
        workouts = conn.execute(
            "SELECT * FROM workouts ORDER BY workout_date DESC, id DESC"
        ).fetchall()
    finally:
        conn.close()
    return render_template("home.html", workouts=workouts)
```

**Step 2: Update POST /log success redirect**

The success redirect already calls `url_for("index")` — confirm it does. If it still says `url_for("new_workout")`, change it back to `url_for("index")`.

### Task 3: Create home.html with sortable workout table

**Files:**
- Create: `templates/home.html`

**Step 1: Create the template**

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="color-scheme" content="light">
  <title>Workout Tracker</title>
  <style>
    html { color-scheme: light; }
    body { font-family: sans-serif; max-width: 960px; margin: 40px auto; padding: 0 16px; background: #ffffff; color: #111; }
    .top-bar { display: flex; justify-content: space-between; align-items: center; margin-bottom: 24px; }
    h1 { font-size: 1.5rem; margin: 0; }
    .btn { padding: 10px 20px; background: #2563eb; color: white; border: none; border-radius: 4px; font-size: 1rem; cursor: pointer; text-decoration: none; }
    .btn:hover { background: #1d4ed8; }
    .flash { background: #d1fae5; color: #065f46; padding: 10px; border-radius: 4px; margin-bottom: 16px; }
    table { width: 100%; border-collapse: collapse; font-size: 0.9rem; }
    th { text-align: left; padding: 10px 12px; background: #f3f4f6; border-bottom: 2px solid #e5e7eb; cursor: pointer; user-select: none; white-space: nowrap; }
    th:hover { background: #e5e7eb; }
    th .sort-arrow { margin-left: 4px; color: #9ca3af; }
    th.sorted-asc .sort-arrow::after { content: " ▲"; color: #2563eb; }
    th.sorted-desc .sort-arrow::after { content: " ▼"; color: #2563eb; }
    th:not(.sorted-asc):not(.sorted-desc) .sort-arrow::after { content: " ⇅"; }
    td { padding: 10px 12px; border-bottom: 1px solid #e5e7eb; }
    tr:last-child td { border-bottom: none; }
    tr:hover td { background: #f9fafb; }
    .empty { color: #6b7280; padding: 24px 0; text-align: center; }
  </style>
</head>
<body>
  <div class="top-bar">
    <h1>Workouts</h1>
    <a class="btn" href="/new">+ New Workout</a>
  </div>

  {% with messages = get_flashed_messages() %}
    {% if messages %}
      {% for msg in messages %}
        <div class="flash">{{ msg }}</div>
      {% endfor %}
    {% endif %}
  {% endwith %}

  {% if workouts %}
  <table id="workout-table">
    <thead>
      <tr>
        <th data-col="0">Type<span class="sort-arrow"></span></th>
        <th data-col="1" class="sorted-desc">Date<span class="sort-arrow"></span></th>
        <th data-col="2">Start<span class="sort-arrow"></span></th>
        <th data-col="3">End<span class="sort-arrow"></span></th>
        <th data-col="4">Duration<span class="sort-arrow"></span></th>
        <th data-col="5">Miles<span class="sort-arrow"></span></th>
        <th data-col="6">Calories<span class="sort-arrow"></span></th>
        <th data-col="7">Avg HR<span class="sort-arrow"></span></th>
      </tr>
    </thead>
    <tbody>
      {% for w in workouts %}
      <tr>
        <td>{{ w.workout_type }}</td>
        <td>{{ w.workout_date }}</td>
        <td>{{ w.start_time or '' }}</td>
        <td>{{ w.end_time or '' }}</td>
        <td>{{ w.duration or '' }}</td>
        <td>{{ w.miles if w.miles is not none else '' }}</td>
        <td>{{ w.active_calories if w.active_calories is not none else '' }}</td>
        <td>{{ w.avg_heart_rate if w.avg_heart_rate is not none else '' }}</td>
      </tr>
      {% endfor %}
    </tbody>
  </table>
  {% else %}
  <p class="empty">No workouts logged yet. <a href="/new">Add your first one.</a></p>
  {% endif %}

  <script>
    (function () {
      const table = document.getElementById('workout-table');
      if (!table) return;
      const headers = table.querySelectorAll('th');
      let sortCol = 1, sortAsc = false;

      function cellVal(row, col) {
        return row.cells[col]?.textContent.trim() ?? '';
      }

      function sortTable(col, asc) {
        const tbody = table.tBodies[0];
        const rows = Array.from(tbody.rows);
        rows.sort((a, b) => {
          const va = cellVal(a, col), vb = cellVal(b, col);
          const na = parseFloat(va), nb = parseFloat(vb);
          const cmp = (!isNaN(na) && !isNaN(nb)) ? na - nb : va.localeCompare(vb);
          return asc ? cmp : -cmp;
        });
        rows.forEach(r => tbody.appendChild(r));
        headers.forEach(h => h.classList.remove('sorted-asc', 'sorted-desc'));
        headers[col].classList.add(asc ? 'sorted-asc' : 'sorted-desc');
      }

      headers.forEach((th, i) => {
        th.addEventListener('click', () => {
          if (i === sortCol) { sortAsc = !sortAsc; }
          else { sortCol = i; sortAsc = true; }
          sortTable(sortCol, sortAsc);
        });
      });

      sortTable(sortCol, sortAsc);
    })();
  </script>
</body>
</html>
```
